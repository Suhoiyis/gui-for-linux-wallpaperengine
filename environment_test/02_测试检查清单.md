# GUI测试检查清单

## 1. 用户界面
- [ ] 是否所有按钮都能正常点击？
- [ ] 是否有明显的错误提示和反馈？
- [ ] 界面布局是否符合设计规范？

## 2. 功能性
- [ ] 所有功能模块是否正常工作？
- [ ] 用户输入是否经过验证？
- [ ] 是否能够处理意外情况（如网络断开）？

## 3. 兼容性
- [ ] 在不同操作系统上（如Windows, Linux）是否表现一致？
- [ ] 是否支持不同分辨率和屏幕比例？

## 4. 性能
- [ ] 启动时间是否在合理范围内？
- [ ] 大文件操作时是否有明显延迟？
- [ ] 内存使用是否在可接受范围内？
